はじめに

JavaScriptはWebブラウザ上で動作するプログラミング言語で、ユーザーとのインタラクションやデータ処理を担います。近年ではNode.jsを使ってサーバーサイドでも広く使われています。

基本構文と特徴

// 変数宣言
let name = "Petra";
const age = 20;

// 関数
function greet(name) {
    return `Hello, ${name}`;
}

DOM操作とイベントハンドリング

document.getElementById("myButton").addEventListener("click", () => {
    alert("クリックされました！");
});

非同期処理（Promise / async/await）

// Promise
fetch("https://api.example.com/data")
    .then(response => response.json())
    .then(data => console.log(data));

// async/await
async function getData() {
    const response = await fetch("https://api.example.com/data");
    const data = await response.json();
    console.log(data);
}

モダンJavaScript・TypeScript

ES Modules: import/exportを使ってコードを分割します。

// math.js
export function add(a, b) { return a + b; }

// main.js
import { add } from './math.js';

TypeScript: 静的型チェックによりバグを事前に防止できます。

React / Vue.js

React: コンポーネントベースでUIを構築。仮想DOMを使用。

Vue.js: テンプレート構文が直感的で、学習コストが低め。

テスト（Jest / Mocha）

// Jest の例
test('加算関数', () => {
    expect(add(1, 2)).toBe(3);
});

便利なライブラリ

  • Lodash: 配列やオブジェクトの処理に便利
  • Day.js: 日付操作が簡単に

PWA（Progressive Web Apps）

オフライン対応やホーム画面追加など、アプリのように使えるWebアプリを構築可能。

WebSocketによるリアルタイム通信

const socket = new WebSocket("wss://example.com");
socket.onmessage = (event) => {
    console.log("受信:", event.data);
};

グラフィックス・数理処理

Canvas APIと三角関数を使って描画・アニメーションも可能です。

// 簡単なCanvas描画
const canvas = document.getElementById("myCanvas");
const ctx = canvas.getContext("2d");
ctx.arc(75, 75, 50, 0, Math.PI * 2);
ctx.stroke();

パフォーマンス最適化

  • コードの再利用性向上（関数分割）
  • メモリリーク対策
  • 画像の遅延読み込み（Lazy Loading）
