JavaScript初心者向けチュートリアル集

このページでは、JavaScript初心者のために4つのミニアプリを通して基本的な技術を学べます。

1. おみくじアプリ作成チュートリアル

学習内容： 変数、配列、条件分岐、乱数、DOM操作、イベント処理

完成イメージ

「おみくじを引く」ボタンを押すと、ランダムな運勢（大吉、中吉など）が表示されます。

HTML + CSS

<!DOCTYPE html>
<html>
<head>
  <meta charset="UTF-8">
  <title>おみくじアプリ</title>
  <style>
    #result { font-size: 24px; margin-top: 20px; }
  </style>
</head>
<body>
  <h1>おみくじアプリ</h1>
  <button id="omikujiBtn">おみくじを引く</button>
  <div id="result"></div>
  <script src="script.js"></script>
</body>
</html>

JavaScript (script.js)

const fortunes = ["大吉", "中吉", "小吉", "凶", "大凶"];
const btn = document.getElementById("omikujiBtn");
const result = document.getElementById("result");

btn.addEventListener("click", function() {
  const index = Math.floor(Math.random() * fortunes.length);
  result.textContent = fortunes[index];
});

2. ToDoリスト作成チュートリアル

学習内容： フォーム入力、DOMの生成・削除

HTML

<input type="text" id="taskInput">
<button id="addTask">追加</button>
<ul id="taskList"></ul>

JavaScript (todo.js)

const input = document.getElementById("taskInput");
const button = document.getElementById("addTask");
const list = document.getElementById("taskList");

button.addEventListener("click", function() {
  const taskText = input.value;
  if (taskText === "") return;

  const li = document.createElement("li");
  li.textContent = taskText;

  const deleteBtn = document.createElement("button");
  deleteBtn.textContent = "削除";
  deleteBtn.addEventListener("click", function() {
    list.removeChild(li);
  });

  li.appendChild(deleteBtn);
  list.appendChild(li);
  input.value = "";
});

3. お天気検索アプリ

学習内容： Fetch API、JSONの扱い、非同期処理

OpenWeatherMapのAPIキーが必要です。こちらから取得

HTML

<input type="text" id="city">
<button id="search">検索</button>
<div id="weather"></div>

JavaScript (weather.js)

const API_KEY = "YOUR_API_KEY";
const button = document.getElementById("search");
const input = document.getElementById("city");
const result = document.getElementById("weather");

button.addEventListener("click", () => {
  const city = input.value;
  fetch(`https://api.openweathermap.org/data/2.5/weather?q=${city}&appid=${API_KEY}&units=metric&lang=ja`)
    .then(response => response.json())
    .then(data => {
      const weather = data.weather[0].description;
      const temp = data.main.temp;
      result.textContent = `${city}：${weather}、${temp}℃`;
    })
    .catch(() => {
      result.textContent = "都市が見つかりませんでした。";
    });
});

4. jQuery画像スライダー

学習内容： jQueryの基本、アニメーション

HTML

<div id="slider">
  <img src="image1.jpg" class="active">
  <img src="image2.jpg">
  <img src="image3.jpg">
</div>

CSS

#slider img {
  display: none;
  width: 300px;
}
#slider img.active {
  display: block;
}

JavaScript (slider.js)

let index = 0;
const images = $("#slider img");

setInterval(() => {
  images.eq(index).removeClass("active");
  index = (index + 1) % images.length;
  images.eq(index).addClass("active");
}, 3000);
