JavaScriptとは？

JavaScriptは、ウェブページに動きを与えるプログラミング言語です。HTMLやCSSと組み合わせて、ユーザーと対話する機能（ボタンを押す、情報を表示するなど）を実現できます。 ここでは、JavaScriptの基本を学びながら、実際に体験できるミニアプリも紹介します。

JavaScriptの基礎

変数と定数

let name = "太郎";
const age = 20;
document.getElementById("output1").textContent = `名前: ${name}、年齢: ${age}`;

関数

function sayHello() {
  alert("こんにちは！");
}
sayHello();

条件分岐

let score = 85;
let message = "";
if (score >= 80) {
  message = "よくできました";
} else {
  message = "がんばりましょう";
}
document.getElementById("output3").textContent = message;

イベントとDOM

document.getElementById("demoBtn").addEventListener("click", function() {
  alert("ボタンがクリックされました");
});
